Introduction

Get started with GESPRO.
GESPRO lets you work more collaboratively and get more done.

Enable you to organize and prioritize your projects in a fun, flexible and rewarding way.

Manage your tasks

You only have to add the following items:
  • Task name
  • Estimation in days [at the beginning the estimated is optional while the task where in BACKLOG list ]
  • Priority

Manage your lists

You only have to add the following items:
  • List name [Valid values: BACKLOG, TO-DO, IN PROGRESS, WAITING, DONE]
  • Sprint name
Also you can delete the list, with all the task inside.
You can delete any task
Or modify the estimation if it's in BACKLOG
If you want to move the task to another list, you can do it from task or list
on the navbar. Possible list changes:
  • BACKLOG to-do
  • TO-DO in progress, waiting, done
  • IN PROGRESS waiting, to-do, done
  • WAITING in progress, to-do, done
  • DONE can't move it
